  1.What is the difference between var, let, and const?

Ans: var : Function scoped, can be redeclared and updated.
let : Block scoped, can be updated but not redeclared.
const : Block scoped, cannot be updated or redeclared

2.What is the spread operator (...)?

Ans:The spread operator expands elements of an array or object into individual values.

3.Difference between map(), filter(), and forEach()

Ans:
map : Creates a new array by transforming each element.

filter :  Creates a new array with elements that pass a condition.

forEach : Just loops through the array, does not return a new array.

4.What is an arrow function?

Ans: A short syntax for writing functions in JavaScript.

5. What are template literals?

Ans: Strings written with backticks ( ) that allow variable embedding and multiline text.
